# Marketing Funnel Analysis Dashboard

## Overview

This project was developed as part of my Data Science & Analytics Virtual Internship. The objective of this project was to analyze marketing campaign data and understand how customers move through the conversion funnel.

Using Power BI, I created an interactive dashboard to evaluate campaign performance, conversion rates, customer segments, and communication channels. The analysis helps identify factors that influence customer conversions and provides actionable business insights.

## Dataset

The project uses the Bank Marketing Dataset, which contains information about customer demographics, marketing campaigns, contact methods, and subscription outcomes.

## Data Cleaning

Before analysis, the dataset was reviewed and cleaned using Power Query in Power BI. The following steps were performed:

* Checked for missing values
* Removed duplicate records
* Verified data types
* Reviewed categorical variables
* Prepared the data for analysis and visualization

## Dashboard Features

The dashboard includes:

* Total Contacts KPI
* Total Conversions KPI
* Conversion Rate KPI
* Funnel Analysis
* Conversions by Job Category
* Conversions by Education Level
* Conversions by Marital Status
* Conversions by Previous Campaign Outcome
* Conversions by Month
* Contact Channel Analysis
* Interactive Filters and Slicers

## Key Insights

* The overall conversion rate was approximately 11.3%.
* Cellular communication was the most frequently used contact channel.
* Customers with university degrees showed strong conversion performance.
* Certain job categories contributed more conversions than others.
* Previous campaign outcomes had a noticeable impact on customer conversion behavior.

## Recommendations

* Focus marketing efforts on high-converting customer segments.
* Prioritize communication channels with better conversion performance.
* Re-engage customers with positive previous campaign interactions.
* Improve campaign strategies for lower-performing customer groups.
* Use data-driven targeting to enhance future marketing campaigns.
